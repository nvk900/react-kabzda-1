• git clone path-to-repository - клонировать репозиторий
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить файл для слежения
  • git commit -am "added nrw file index.html" - закомитить, зафиксировать изменения и дать им комментарий
  • git push - отправить все закомиченные изменения в репозиторий на сервере на гитхаб
  • git pull - забрать изменения
